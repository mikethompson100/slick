Example 1 - Basic Carousel

6 slides

Example 2 - infinite and dots

6 slides


$('#carousel2').slick({
    slidesToShow: 2, 
    slidesToScroll: 2, 
    dots: true,
    infinite: false
});

Example 6 - asNavFor (Carousels 6 & 7)

6 slides

Carousel 6
Carousel 7
    
$('#carousel6').slick({
  asNavFor: "#carousel7",
  dots: true,
  arrows: false
}); 
$("#carousel7").slick({
  slidesToShow: 3,
  asNavFor: "#carousel6"
});

Example 8 - asNavFor (Carousels 8 & 9 with centerMode/focusOnSelect)

6 slides

Carousel 8
Carousel 9

    $('#carousel8').slick({
        asNavFor: "#carousel9",
        arrows: false,
        fade: true
    }); 

    $("#carousel9").slick({
        asNavFor: "#carousel8",
        focusOnSelect: true, (click on slide)
        centerMode: true,
        slidesToShow: 3             
    });

 (slidesToShow needs to be an odd # to center)

Example 10 - appendArrows/responsive

6 slides

Using appendArrows to target specified divs for arrow section rendering.
Also designate separate locations for both mobile and desktop

Mobile arrows location:
Desktop arrows location: 


  $("#carousel10").slick({
    slidesToShow: 3,
    arrows: true,
    appendArrows: '.appendArrowsDesktop',
    responsive: [
      {
        breakpoint: 768,
        settings: {
          slidesToShow: 3,
          arrows: true,
          appendArrows: '.appendArrowsMobile'
        }
      }
    ]
  });

Example 11 - prevArrow/nextArrow and appendDots

6 slides

Using prevArrow/nextArrow to target specified divs for unique arrow image or text rendering.

Also using appendDots to target specific div above the carousel instead of below it.



  $('#carousel11').slick({
    slidesToShow: 3,
    dots: true,
    appendDots: '.appendDots',
    arrows: true,
    prevArrow: '.previous',
    nextArrow: '.next'
  });

cubic-bezier(x1,y1,x2,y2)

      });
